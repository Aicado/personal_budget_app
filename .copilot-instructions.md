# Copilot Instructions

- Use `uv` to run Python commands in this project.
- Recreate the virtual environment with:
  ```bash
  uv venv --clear .venv
  uv sync
  ```
- Run Python scripts through uv to ensure the managed environment is used:
  ```bash
  uv run python <script.py>
  ```
- Run the backend with uv in development mode:
  ```bash
  uv run -m uvicorn src.backend.main:app --reload --host 0.0.0.0 --port 8000
  ```
- The Python backend uses `polars`, `pyarrow`, and `duckdb` instead of `pandas`.
- Always prefer `uv run` when executing Python code for validation and tests.
